# Agma.JS

## Installation

```shell
# if you use NPM
npm install --save agma.js
# or, if you use Yarn
yarn add agma.js
```

## Importing

```javascript
// CommonJS
const Agma = require('agma.js');
// ES6
import * as Agma from 'agma.js';
```
